#Smart Contract based voting system which let voters elect candidates in an election. 
This is the upcoming way of casting vote where all votes will be stored in the blockchain, which makes the system tamper proof and fully transparent. Voting tokens are issued to everybody who is eligible to vote and let them vote. Technology Stack - Solidity, Truffle, Ganache, NodeJs, Bootstrap.
